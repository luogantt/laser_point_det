# bin-to-pcd-python
Converting bin files scanned by LIDAR of autonomous vehicles into pcd files for further processing

## Requirements

The code requires *open3d* 

## convert `.bin` to `.pcd`

usage:

```bash
python bin2pcd.py binFileName, pcdFileName
```

for example:

```bash
python bin2pcd.py 000002.bin 2.pcd
```



## Dataset

KITTI, Sementic3D
